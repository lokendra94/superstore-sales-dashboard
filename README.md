\# Superstore Sales Dashboard



\## Project Overview



This project analyzes sales performance using the Sample Superstore dataset. The dashboard was built in Power BI after cleaning and preparing the data in Excel.



\## Tools Used



\- Microsoft Excel

\- Power BI

\- Git

\- GitHub



\## Dashboard Pages



\- Sales Overview

\- Geographic Analysis

\- Customer Analysis

\- Shipping Analysis



\## Key Features



\- KPI Cards

\- Interactive Year Filter

\- Monthly Sales Trend

\- Geographic Sales Analysis

\- Customer Segment Analysis

\- Shipping Performance Analysis
